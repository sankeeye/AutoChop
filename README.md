Wat doet het
Truck rijdt pulserend vooruit met W (gas-tijd 0.20s, wacht-tijd 3.20s) en spamt continu Q om te chop'en. Draaien doe je zelf met A/D.

Installatie
Download AutoChop.zip hieronder
Pak uit naar bijv. je bureaublad
Rechtsklik start.bat → "Als administrator uitvoeren"
Eerste keer: Python wordt automatisch geïnstalleerd
Hotkeys
F6 — AUTO aan/uit
PgUp / PgDn — wacht-tijd tunen (langzamer / sneller)
F3 / F4 — gas-tijd tunen
F8 — script stoppen
Vereisten
Windows 10/11
Internet (eenmalig voor Python install)
Admin rechten
